# Java Console Calculator

## Objective
Build a basic calculator that runs in the Java console, allowing users to perform addition, subtraction, multiplication, and division using a simple text-based interface.

---

## Tools & Technologies
- Java (JDK 8 or above)
- VS Code or IntelliJ Community Edition
- Terminal / Command Prompt

---

## Project Structure
ConsoleCalculator.java — Main Java source file  
README.md — Project documentation

---

## How to Run
1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Compile the Java file:  
   javac ConsoleCalculator.java
4. Run the program:  
   java ConsoleCalculator

---

## Features
- Simple menu-based interface
- Performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with division-by-zero error handling)
- Allows multiple calculations until the user chooses to exit

---

## Learning Outcome
- Practice Java syntax and console I/O
- Create and use methods
- Use loops, switch-case, and conditional logic
- Handle user inputs effectively
